# Cyberpunk Hyprland Rice

This is my personal Hyprland rice, built on Gentoo Linux.

It has gone through a few different styles over time. It started as a Catppuccin setup, then I experimented with a macOS-inspired look, and eventually settled on a cyberpunk aesthetic.

## Notes

This repository is **not intended to be plug-and-play**.

Many configuration files contain machine-specific paths and settings, so you'll likely need to edit them to match your own system. Don't copy everything blindly—take some time to understand what each configuration does and customize it for your setup.
